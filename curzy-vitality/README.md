# Curzy Vitality Dashboard

This is the "Nexus Dashboard" for the Curzy Vitality (CV) system, now officially renamed to **Curzy Vitality**.

## Overview
A high-performance React dashboard that visualizes system activity tracked by the Sentinel module.

## Setup
1. Ensure the Sentinel backend is running on port 3005.
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
